# Nonlinear-System-Identification-and-Rotor-Driven-Pendulum-Modelling-using-Neural-Networks
Hybrid physics-informed machine learning for nonlinear system identification and MPC control of a rotor-actuated pendulum — combining Neural ODEs and residual gray-box learning (Newton-Euler + NN correction) with RK4-based Model Predictive Control, targeting real plant deployment.
